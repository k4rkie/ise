# ise

ise is a isometric topdown game engine written in C using [Raylib](https://github.com/raysan5/raylib) as a purely hobby project.

> [!WARNING]
> Still in super early stages

### Usage

#### Clone the repo

```bash
git@github.com:k4rkie/khoj.git
```

#### Run make

```bash
mkdir build
make run
```
